#  NoteSharePro – Engineering Notes Marketplace

NoteSharePro is a modern frontend-based Engineering Notes Marketplace built using Vue 3 and Tailwind CSS.  
It allows students to browse, upload, and simulate selling engineering notes across multiple domains.

---

## Features

### Login System
- Simple email-based login (LocalStorage session)
- Persistent login state

###  Browse Notes
- Category-based filtering
- Search by subject or topic
- Premium & Free tag system
- Responsive card layout

###  Wallet Simulation
- Track total earnings
- Premium notes earn ₹49 per purchase (simulated)

###  Upload Notes
- Add new notes with:
  - Title
  - Category
  - Premium toggle
- Instantly appears in dashboard

###  Smart Filters
- Category tabs
- Real-time search functionality
- Reset filter option

---

## Tech Stack

- HTML5
- Tailwind CSS
- Vue.js 3 (CDN)
- Font Awesome
- LocalStorage (for data persistence)
